# my-tools
自分用の小さなツール集

## bbhd

- 指定したディレクトリをホスティングするwebサーバー

## bd2p

- "GNU barcode"と"imagemagic"を使いjanコードを生成

## c2h

- カンマ区切りのテキストファイルをhtmlのtableに変換

## c2x

- カンマ区切りのテキストファイルをxmlに変換

## cpugv

- cpuガバナーの調整

## csrch

- 対話形式でcsvファイル内を検索

## sands,sandv

- 自分のデスクトップPCの設定用コマンド

## tbug

- "/tmp/"ディレクトリにパイプの途中経過を保存
